Solution to question #1 of the Bitnine Apache AGE Internship Coding Test

Made by Eduardo Monteiro de Sousa using the Vim text editor and the GCC
compiler on a x86_64 Fedora 37 GNU/Linux system.

How to compile:
1 - Make sure the directory where this file resides is the
    current working directory.

2 - Compile the project with the following command:
    gcc -O3 -Wall -std=c17 src/main.c -o question1

3 - Make sure you have the right permissions and run the application:
    ./question1
